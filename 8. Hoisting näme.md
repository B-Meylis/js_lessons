## Hoisting näme?

**Hoisting** — bu JavaScript-de üýtgeýjileriň we funksiýalaryň (declarations) kod ýerine ýetirilmezden ozal scope-yň başyna “göterilmek” mehanizmidir.

**Möhüm❗️** Göterilýäni diňe yglan edilenler(объявления), berlen bahalar (инициализации) däl! Şeýle-de kod hakykatda ýerini üýtgetmeýär — bu diňe ýerine ýetiriş kontekstiniň (execution context) döredilen wagtynda emele gelýän hereketdir.

**Hoisting aslynda nähili işleýär**
JavaScript ýerine ýetiriş kontekstini (global ýa-da funksiýa konteksti) döredende iki tapgyrdan geçýär:
1. **Dörediş tapgyry (Creation Phase)** — JavaScript kody skanirleýär we üýtgeýjiler hem-de funksiýalar üçin ýat bölüp berýär.

2. **Ýerine ýetiriş tapgyry (Execution Phase)** — kod setir-setir ýerine ýetirilýär.

**1-nji ädim: Hoisting(Поднятие)**
Еngine yglan edilenleri  kontekstiň başyna “göterýär”:
```js
// Biz ýazýarys:
console.log(x);
var x = 10;
function test() {}
// Engine-niň parsing etme wagtynda görşi:
var x;               // ← yglan edilen göterildi
function test() {}   // ← funksiýa tutuşlygyna göterildi

console.log(x);      // undefined (baha heniz berilmedi)
x = 10;              // ← baha bermek öz ýerinde galdy
```

**2-nji ädim: Scope Chain(Цепочка областей видимости
)**
**Scope Chain** — JavaScript-iň belli bir ýerde duran kodyň haýsy üýtgeýjilere we funksiýalara elýederlidigini(доступно) kesgitleýän mehanizmidir. Bu, leksiki gurşawlara (lexical environments) salgylanmalaryň zynjyry bolup, ýerine ýetiriş wagtynda däl-de, kod ýazylan wagty emele gelýär.
Engine her üýtgeýjiniň nirede ýerleşýändigini ýatda saklaýar:
```js
let global = "🌍";

function outer() {
  let outerVar = "📦";

  function inner() {
    let innerVar = "🎁";

    console.log(global);   // Gözleýär: inner → outer → global
    console.log(outerVar); // Gözleýär: inner → outer ✅
    console.log(innerVar); // Gözleýär: inner ✅
  }

  inner();
}

outer();
```



**Engine üýtgeýjini nähili gözleýär?(Wizual)**

```text

Haçan-da JS Engine inner funksiýanyň içinde "outerVar" görse:

1. Ilki inner() funksiýasynyň ýadyna (Local Storage) seredýär:
   ┌─────────────────────┐
   │ innerVar: "🎁"      │ ← outerVar? Ýok
   └─────────────────────┘
             ↓
      (Parent scope salgysy)

2. Soň outer() funksiýasynyň ýadyna seredýär:
   ┌─────────────────────┐
   │ outerVar: "📦"      │ ← ✅ Tapdy!
   └─────────────────────┘
             ↓
      (Parent scope salgysy)

3. (Eger tapmadyk bolsa) global scope-a seredýär.
   ┌─────────────────────┐
   │ globalVar: "🌍"     │
   └─────────────────────┘
```
