Kommentarýalar - bir setirli: *`//`* bilen başlanýar, ýa-da köp setirli: *`/*... */`*. Adatça olary koduň nädip we näme üçin işleýändigini düşündirmek üçin ulanýarys.

*2. Çylşyrymly kody funksiýalara bölüň*
❌️
```js

for (let i = 2; i < n; i++) {
    // i-niň ýönekeý san bolandygyny barlaýarys
    let isPrime = true;
    for (let j = 2; j < i; j++) {
        if (i % j == 0) {
            isPrime = false;
            break;
        }
    }
    if (isPrime) alert(i);
}
```

✅️
```js
function showPrimes(n) {
    for (let i = 2; i < n; i++) {
        if (isPrime(i)) alert(i);
    }
}

function isPrime(n) {
    for (let i = 2; i < n; i++) {
        if (n % i == 0) return false;
    }
    return true;
}
```
