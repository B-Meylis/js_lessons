## JS use strict

*"use strict"* – bu JavaScript-i berk (strict) režim bilen işlemäge kömek edýär, ýalňyşlyklary şol wagt ýüze çykarýar we howply hereketleriň öňüni alýar.

🎯 **Näme üçin gerek?**

Mysal üçin, adaty režimde:
```js
x = 10; // Global üýtgeýän döredilýär
```


Ýöne *"use strict"* bilen:
```js
"use strict";

x = 10; // ❌ ReferenceError: x is not defined
```

**Strict režimi nädip işletmeli?**

**1. Tutuş faýl üçin**
```js
"use strict";

// Aşakdaky ähli kod strict režimde işleýär
let name = "Вася";

function hello() {
    console.log("Привет, " + name);
}
```

**2. Diňe bir funksiýanyň içinde**
```js
function strictFunction() {
    "use strict";

    let x = 10;
}

function normalFunction() {
    y = 20; // Adaty režimde global üýtgeýji dörediler
}
```
Bu mysalda strict režim diňe *strictFunction()* üçin işleýär.

Strict režimi näme edýär?  
✅️ Üýtgeýänler *let, const* bilen döredilmeli  
✅️ Üýtgeýjlänleri *delete* bilen pozup bolmaýar  
✅️ Dublikat parametrler gadagan  
✅️ *this = undefined*  

ES6 we ondan ýokarky wersiýalarda:  
● Klasslar(*class*) - awtomatiki strict režimde  
● Modullar (.mjs ýa-da <script type="module">) – awtomatiki  
● Arrow funksiýalar - strict režimden miras alýarlar(наследуют)
